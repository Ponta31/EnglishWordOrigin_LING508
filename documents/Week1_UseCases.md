# Use Cases

## Use Case ①: Search for Word Origin *(This project)*
- **Input:** An English word (e.g., `revoked`)  
- **Output:**  
  - Lemma form  
  - Meaning (in Japanese)  
  - Pronunciation (IPA)  
  - Etymology (in Japanese)  
  - Morphemes (e.g., `re- = again`, `vocare = to call`)  

---

## Use Case ②: Mnemonic in Japanese for an English Word *(planned feature)*
- **Input:** An English word (e.g., `abrupt`)  
- **Output:**  
  - Meaning  
  - Pronunciation  
  - A mnemonic in Japanese for the English word (e.g., `油ぷっと吹き出す`)
